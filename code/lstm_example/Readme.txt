This is an example for the ESSLLI course "Analyzing the Cognitive Plausibility of Language Models" taught by Lisa Beinborn (Vrije Universiteit Amsterdam), Nora Hollenstein (University of Copenhagen), and Willem Zuidema (University of Amsterdam).

The code has been adapted from https://github.com/shivam5992/language-modelling by Shivam Bansal
It requires the packages keras and numpy.

The data is extracted from http://www.gutenberg.org/cache/epub/12003/pg12003.txt
(chapter I-III).
